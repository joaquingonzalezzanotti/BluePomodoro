# BluePomodoro 🚀

Tu herramienta definitiva de productividad con la técnica Pomodoro e IA.

## 🛠️ Despliegue (Vercel o Firebase)

Puedes publicar esta aplicación de forma gratuita siguiendo estos pasos:

### 1. Subir a GitHub
1. Crea un nuevo repositorio en tu cuenta de GitHub.
2. Sube el código de este proyecto.

### 2. Desplegar
- **Vercel**: Importa el repositorio desde el panel de Vercel.
- **Firebase App Hosting**: Conecta tu repositorio de GitHub directamente desde la consola de Firebase en la sección "App Hosting".

### 3. Configuración de Seguridad (OBLIGATORIO)
Para que el inicio de sesión con Google funcione en producción:
1. Ve a la [Consola de Firebase](https://console.firebase.google.com/).
2. Entra en **Authentication** > **Settings** > **Authorized domains**.
3. Haz clic en **Add domain** y añade el dominio de tu app desplegada (ej: `blue-pomodoro.vercel.app` o `tu-app.web.app`).

## 🧠 Funcionalidades de IA
- **Desglose de Tareas**: Divide tareas complejas en pasos accionables.
- **Priorización Inteligente**: Organiza tu día basándose en niveles de energía y fechas límite.
